    # 以下配置项已整合为设置弹窗

    1. 关闭点击生词发音、复制；双击未标注单词获取网络发音、复制；默认开启

      is_voc_copy_explain = false;
      1

    2. 主窗口 点击生词、双击时 不再发音 右侧单词释义列表正常发音；默认发音

      is_voc = false;
      1

    3. 主窗口 点击生词、双击时 关闭自动复制；默认复制

      is_copy = false;
      1

    4. 主窗口 点击生词时 右侧 不 跳转释义详情；默认跳转

      is_explain = false;
      1

    5. 主窗口 双击未标注的 不再标注；默认标注

      is_select_mark = false;
      1

    6. word-filler 双击选中状态; 默认不选中

      is_dbl_select = false;
      1

    7. 文中 不标注出重复的单词；默认标注

      is_dup = false;
      1

    8. 开启原生右键菜单 默认屏蔽

      is_menu_prevent = false;
      1

    9. 方向键 前进步数 默认5；

      step = 5;
      1

    10. 等级选择框 选择默认后 生词弹窗【M按钮】是否要保存等级排序 ；默认保存

    is_also_mark_save = true;
    1

    1. 生词弹窗、右侧单词释义列表 右键单击时， 文中的等级标注 是否去除；默认去除

      is_mark_del = true;
      1
