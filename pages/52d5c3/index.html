    # 2、使用

    1. 点击新标签或新窗口转至”home页“

      新标签：新标签页打开home页

      新窗口：新窗口打开home页

      添加：可选择分组

      微：添加至微应用

    2. 添加分组的方式

      1、从提示中创建

      2、右上角菜单中

      3、某个分组的右键菜单中（添加至改分组后边）

    3. 添加网站的方式

      1、popup页面添加

      2、某分组的右键菜单添加（添加至该分组末尾）

      3、某一项书签的右键菜单（添加至该书签之后）

    4. 将“home页”设置为“主页”（不重要，根据个人喜好，可跳过）

      • Edge浏览器：设置-开始、主页和新建标签页

      • 填自己电脑上该扩展home页的网址，不同电脑不一样

      • 建议将该导航页插件主页作为应用安装到本地【Edge浏览器有此功能】【右上角··· — 应用 — 将此站点作为应用安装】
