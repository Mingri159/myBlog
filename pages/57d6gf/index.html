更新日志

# v1.6.7

2023.3.25

  • 更改，【导入备份文件】对话框下的【不覆盖】改为【合并】；
  • 优化，导入【备份文件】时，不直接覆盖，而是合并新增的分组和书签； - 避免PC端和移动端不一致的数据在导入时一并被覆盖；
  • 修复，表单校验残留的问题；
# v1.6.6

2023.3.18

  • 移除，【设置】-【无】壁纸模式；
  • 移除，【导出】分组备份文件中【分组+设置】项；
  • 移除，【历史壁纸】菜单中【复制链接】项；
  • 修复，【历史壁纸】菜单错位问题；
  • 优化，【历史壁纸】页面；
  • 优化，【壁纸裁剪】页面；
  • 优化，【回收站】页面；
  • 优化，分组内书签布局；
  • 增加，分组序号，可在【设置】中关闭；
  • 增加，分组图标，可在【设置】中关闭；
  • 增加，书签点击波纹效果；
  • 更改，【导入】不再处理【设置】的相关数据；
# v1.6.5

2023.2.28

  • 支持本地壁纸裁剪；
  • 修复所有右键菜单错位问题；
  • 优化UI背景显示效果；
  • 移除图标校验：手动添加书签时，【图标】不是必填项；

# v1.6.4

2022.10.12

  • 添加 快捷键；

  • 添加 背景模糊

  • 修复 对话框、设置页面、返回顶部、图标菜单等随页面滚动问题；

  • 修复 事件穿透问题；

  • 修复 导入分组时 不显示导入按钮

  • 优化 输入框的校验逻辑；

  • 优化 一些细节；
