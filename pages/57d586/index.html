    # 一、可能遇到的问题（不影响Edge扩展商店的）

    # 1、向1.6.0版本数据迁移

    • 问题：1.6.0版本导入1.5.3版本备份后，页面卡死

    • 原因：

      1.6.0版本：

    • 解决方法（1）：

      • 复制备份文件.json生成备份文件 - 副本.json
      • 记事本打开备份文件 - 副本.json
      • 删除"settings": {xxxxxxxxxxxxx}
      • 1.6.0版本直接导入即可

    • 解决方法（2）：

      • 需要文件 1.5.2版本option.js

      • 步骤 1、将1.5.3版本option.js替换为1.5.2版本option.js，刷新页面 2、导出备份（1.5.2版本备份文件） 3、安装1.6.0版本 4、1.6.0版本直接导入上述的1.5.2版本备份文件即可，壁纸等需要重新设置
